# Sistema de Detracciones SPOT

## Despliegue en Railway.app (GRATIS)

1. Crear cuenta en https://railway.app
2. New Project → Deploy from GitHub repo
3. Subir esta carpeta a GitHub primero:
   - Instalar Git: https://git-scm.com
   - `git init`
   - `git add .`
   - `git commit -m "inicial"`
   - Crear repo en github.com y seguir instrucciones
4. En Railway: conectar el repo → Deploy
5. Listo. URL pública automática.
